name: Walkthrough process for review findings
description: Validated interactive walkthrough approach — point-by-point with terse summaries, inline corrections on ACCEPTED, recap at end
type: feedback
---

Interactive point-by-point walkthrough of review findings works well: present each finding with severity, summary, opinion, then 4 choices (ACCEPTED/NOTED/DEFERRED/REJECTED). Apply corrections inline immediately after each ACCEPTED.

**Why:** The user responds quickly (often just "y" or "suppr") when the finding is well summarized and the opinion is clear. No need to over-explain — get to the point.

**How to apply:**
- Present: ID, severity, 2-3 line summary, audit recommendation, personal opinion, choice
- For NOTED without action: move directly to the next finding without waiting for additional confirmation
- For ACCEPTED: correct immediately, confirm, move on
- Update audit reports with decisions at the end of the walkthrough (summary table)
- Group cross-referenced findings to avoid duplicates
