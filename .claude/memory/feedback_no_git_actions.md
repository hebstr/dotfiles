name: No git actions
description: User manages git operations (commit, push, etc.) themselves — never propose or execute them
type: feedback
---

Do not propose or execute git commits, pushes, or other git write operations.

**Why:** The user wants to keep full control over their git workflow.

**How to apply:** Stop at the code/file changes. Do not offer to commit, push, or create PRs unless the user explicitly asks.
